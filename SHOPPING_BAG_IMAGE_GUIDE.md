# Shopping Bag - Инструкция по добавлению изображений

## Названия файлов и цвета

Все изображения должны быть в формате **AVIF** с точными названиями:

### Список файлов:

| Цвет | Hex код | Название файла |
|------|---------|----------------|
| Black | `#000000` | `black-1.avif` |
| Grey | `#d1d1d1` | `grey.avif` |
| Midnight Blue | `#3d4f6b` | `midnight-blue.avif` |
| Natural | `#f8f4e8` | `natural.avif` |
| White | `#FFFFFF` | `white-1.avif` |
| White (alt) | `#FFFFFF` | `white-2.avif` *(опционально)* |

## Куда добавлять файлы

### 1. Для секции Products (витрина продуктов)

**Путь:** `/public/images/products/shopping-bag/`

**Требования:**
- Формат: AVIF
- Размер: 1200 × 1600 px
- Соотношение: 3:4 (портрет)
- Фон: Чистый, профессиональная фотография продукта

**Файлы:**
```
/public/images/products/shopping-bag/
├── black-1.avif
├── grey.avif
├── midnight-blue.avif
├── natural.avif
├── white-1.avif
└── white-2.avif (опционально)
```

### 2. Для Configurator (конфигуратор дизайна)

**Путь:** `/public/images/configurator/shopping-bag/`

**Требования:**
- Формат: AVIF
- Размер: 1000 × 1000 px
- Соотношение: 1:1 (квадрат)
- Фон: Прозрачный или белый
- Центр: Свободное место для дизайна пользователя

**Файлы:**
```
/public/images/configurator/shopping-bag/
├── black-1.avif
├── grey.avif
├── midnight-blue.avif
├── natural.avif
├── white-1.avif
└── white-2.avif (опционально)
```

## Важные замечания

1. **Названия файлов должны быть точными** - система ищет именно эти имена
2. **Формат AVIF обязателен** - для оптимальной производительности
3. **Размеры важны** - соблюдайте указанные пропорции
4. **Для конфигуратора** - оставьте четкую область в центре для размещения дизайнов

## Проверка

После добавления файлов:
1. Перезагрузите страницу сайта
2. Перейдите в секцию **Products**
3. Выберите **Shopping Bag**
4. Проверьте все 5 цветов
5. Перейдите в **Configurator**
6. Выберите **Shopping Bag** и проверьте все цвета
