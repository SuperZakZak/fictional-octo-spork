# Shopping Bag Product Images

This folder should contain product images for the Shopping Bag in different colors.

## Required Images

Place images with the following names (AVIF format):

- `black-1.avif` - Shopping Bag in black color
- `grey.avif` - Shopping Bag in grey color
- `midnight-blue.avif` - Shopping Bag in midnight blue color (#3d4f6b)
- `natural.avif` - Shopping Bag in natural/off-white color (#f8f4e8)
- `white-1.avif` - Shopping Bag in white color
- `white-2.avif` - Shopping Bag in white color (alternative)

## Image Specifications

- **Format**: AVIF (preferred), WebP, or PNG
- **Aspect Ratio**: 3:4 (portrait)
- **Recommended Size**: 1200x1600px
- **Background**: Clean, professional product photography
- **Quality**: High resolution for product showcase

## Color Codes

- Black: `#000000` → `black-1.avif`
- Grey: `#d1d1d1` → `grey.avif`
- Midnight Blue: `#3d4f6b` → `midnight-blue.avif`
- Natural: `#f8f4e8` → `natural.avif`
- White: `#FFFFFF` → `white-1.avif` or `white-2.avif`

## Notes

The system will automatically try different formats (AVIF → WebP → PNG) if the preferred format is not available.
